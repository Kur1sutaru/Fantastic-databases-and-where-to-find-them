### Case study: Exploratory ACE2 analysis using multi-omic web tools

Alternative splicing:
1. ASPicDB - ( http://srv00.recas.ba.infn.it/ASPicDB/ )

1. CCLE - (https://portals.broadinstitute.org/ccle)
